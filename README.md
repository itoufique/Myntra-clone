# Myntra Clone

A responsive Myntra Clone built using HTML and CSS.  
This project replicates the basic UI and layout of the Myntra website including navigation bar, banners, product sections, and footer.

## Features

- Responsive Navigation Bar
- Hero Banner Section
- Shopping Categories
- Product Cards
- Footer Section
- Clean UI Design
- Responsive Layout

## Technologies Used

- HTML5
- CSS3
